# One Dark Pro AMOLED

Firefox theme with OLED-optimized true black background, built on One Dark Pro's color palette.

## Install

[Get it on addons.mozilla.org](https://addons.mozilla.org)

## Colors

| Element | Hex |
|---------|-----|
| Frame / NTP / Sidebar | `#000000` |
| Toolbar / Popup / Cards | `#0a0a0a` |
| URL bar | `#101010` |
| Active tab / Focus | `#181818` |
| Tab line (accent) | `#61afef` |
| Loading indicator | `#56b6c2` |
| Icons | `#abb2bf` |
| Icon attention | `#e06c75` |
| Text (primary) | `#d7dae0` |
| Text (secondary) | `#abb2bf` |
